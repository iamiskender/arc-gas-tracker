# Arc Network AI Context & Build Guide

## Overview
Arc Network is a decentralized application (dApp) ecosystem designed for high-performance Web3 operations. It integrates closely with the Base ecosystem, utilizes smart contracts for automated actions, and leverages platforms like Polymarket for predictive market actions.

## Development Standards
- **Network:** Base (Ecosystem Integration)
- **Tokens:** USDC, ETH (Gas fees)
- **Key Features:** Low latency, secure smart contract execution, no-code AI deployment options.
- **Deployment Goal:** Creating functional, simple web interfaces interacting with decentralized backends.
